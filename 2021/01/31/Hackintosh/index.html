Created2021-01-31|Updated2021-02-05|Tech

As the saying goes, there are three focuses for geeks,

  • NAS
  • Soft Router
  • Hackintosh

Nowadays, YDJSIR proudly declares that YDJSIR has attempts on all the three aspects. YDJSIR’s experience of NAS and soft router has been introduced in OpenWRT with IPv6 in CERNET and Proxy , so YDJSIR won’t talk about them here. The passage is about YDJSIR’s Hackintosh device.

Status of YDJSIR’s Hackintosh

VDA decode may be effected by YDJSIR’s USB driver, but YDJSIR don’t care it as YDJSIR haven’t meet any difficulties in YDJSIR’s daily use.

EFI files from

https://github.com/stutter0718/ASUS-TUF-B360M-OC

As these EFI files are prepared for Big Sur, you must edit the config.plist if you want to use Catalina or other versions of macOS. Here is a tutorial.

https://macoshome.com/hackintosh/hcourse/3828.html

![WeChat Image_20210201161856](https://oss.ydjsir.com.cn/GitPages/Hackintosh/WeChat Image_20210201161856.jpg)

YDJSIR also has tested EFI from these GitHub repositories as well, but end with severe bugs.

However, they remind me to change settings of my BIOS.

  • Advanced —> Onboard Devices Configuration-Serial Port Configuration Serial Port —> Off
  • Advanced —> USB Configuration-XHCI Hand-off —> Enabled
  • Secure Boot -> OS Type -> Other OS
  • Advanced —> PCH configuration-IOAPIC 24-119 Entries —> disabled
  • CFG-LOCK -> disabled
  • Advanced -> Advanced PCH Configuration -> system time and alarm source -> Legacy RTC

Actual name of these items in your BIOS may vary in different firmware versions. YDJSIR suggest you to search them in the BIOS’s GUI directly (in English).

https://github.com/BDZNH/Hackintosh-EFI

https://github.com/jsfelix/EFI-backup

https://github.com/cuiyijiee/MyBlackAppleEFI

Feelings

APPLE has won a hearts of many loyal customers based on price and specialized software ecosystem. macOS makes me feel that it is a system extremely suitable for productivity. No matter its extremely detailed permission management (just the same as iOS) or its extremely smooth and beautiful UI experience excites YDJSIR. It is a system with high security that is smooth and easy to use without extra work. At the same time, the existence of the App Store in the closed-loop ecology (especially an App Store with acceptable access speeds in mainland China) also provides more and better genuine software. Their deep integration with macOS has surprised YDJSIR. For a long time, the main OS of computers of YDJSIR is obviously still Windows and Linux (including WSL2). However, obviously it does not rule out that one day YDJSIR will adopt MacBook Pro as my productivity tool after having a fixed and powerful device.
